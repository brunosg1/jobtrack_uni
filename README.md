# jobtrack

O JobTrack Uni ajuda estudantes a organizar seu portfólio e acompanhar de forma eficiente as candidaturas para vagas de estágio. Na primeira execução, ele entrega o valor rápido (criar card de vaga) antes de coletar dados pessoais, guiando o usuário por um onboarding curto, leitura de políticas e consentimento explícito.
